# udacity-react-user-poll

React User Poll is s responsive and mobile-ready app that presents the user with a series of questions.

It's built with React, Redux, React Router, & Redux Thunk.

This app is the second of three projects required for [Udacity's React Nanodegree program](https://www.udacity.com/course/react-nanodegree--nd019).

## Installation

Clone the repository, change directories, and use NPM to install the dependencies.

```bash
$ git clone https://github.com/nguyenson2012/udacity-react-user-poll
$ cd udacity-react-user-poll
$ npm install
```

## Usage

The project can be run with

- `npm start`

The project can be viewed in the browser at

- [http://localhost:3000](http://localhost:3000)
